## 1) 앱 기본 정보
- App Name: `WebP Master for ChatGPT`
- Category 제안: `Productivity` 또는 `Developer Tools`

## 2) 스토어 한 줄 소개 (KR)
- ChatGPT에서 만든 이미지를 바로 WebP로 경량화해, 게시 가능한 파일로 빠르게 바꿔주는 변환 앱

## 3) Short Description (EN)
- Convert ChatGPT-generated images to lightweight WebP files instantly, with practical quality presets for publishing workflows.

## 4) 상세 설명 (KR)
WebP Master for ChatGPT는 이미지 생성 후 최적화 단계를 ChatGPT 대화 흐름 안에서 바로 처리하도록 만든 앱입니다.

사용자는 ChatGPT에서 만든 이미지를 업로드해 WebP로 변환하고, 목적(블로그 썸네일, 콘텐츠 배너, 상품 이미지)에 맞는 품질 옵션으로 파일 용량을 줄일 수 있습니다. 여러 이미지를 한 번에 처리한 뒤 ZIP으로 내려받을 수 있어 게시 전 작업 시간을 줄여줍니다.

이 앱의 핵심 가치는 단순 포맷 변환이 아니라, 생성 -> 최적화 -> 게시의 실무 흐름을 끊기지 않게 연결하는 데 있습니다.

## 5) Long Description (EN)
WebP Master for ChatGPT is built for the post-generation step: optimizing images for publishing without leaving your ChatGPT workflow.

You can take generated images, convert them to WebP, and apply practical quality settings depending on your use case (blog thumbnails, content banners, product visuals). The app supports batch conversion and ZIP download to speed up real publishing tasks.

Its value is not just file conversion. It closes the workflow gap between image creation and production-ready delivery.

## 6) 핵심 가치 포인트 (심사/소개 공용)
- ChatGPT 생성 이미지를 즉시 경량화
- 용도별 품질 선택으로 의사결정 시간 단축
- 배치 변환 + ZIP 다운로드로 실무 효율 향상
- 게시 전 최적화 워크플로우를 대화 맥락에서 연속 처리

## 7) 개인정보/데이터 처리 문구 (KR)
- 이미지 변환은 브라우저 환경에서 처리되며, 사용자 이미지를 별도 저장하거나 모델 학습 데이터로 사용하지 않습니다.

## 8) Privacy/Data Handling (EN)
- Image conversion runs in the browser. User images are not persistently stored by this app and are not used as model training data.
