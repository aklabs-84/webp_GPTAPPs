# ⚡ WebP Master (초고속 이미지 변환기)

> **서버 업로드 없이 브라우저에서 안전하고 빠르게 이미지를 WebP로 변환하는 도구입니다.**

WebP Master는 디자이너, 개발자, 블로거를 위한 최적의 이미지 최적화 솔루션입니다. 별도의 프로그램 설치 없이 웹 브라우저만으로 JPG, PNG 등의 이미지를 고효율 WebP 포맷으로 변환하여 웹사이트 로딩 속도를 획기적으로 개선할 수 있습니다.

## ✨ 주요 특징

- **🔒 100% 프라이버시 보호**: 모든 변환 과정이 **사용자의 브라우저(Client-side)** 내에서만 이루어집니다. 이미지가 외부 서버로 전송되지 않아 민감한 데이터도 안심하고 사용할 수 있습니다.
- **🚀 압도적인 속도**: 최신 브라우저의 기술을 활용하여 대량의 이미지도 순식간에 변환합니다.
- **📂 편리한 일괄 처리**: 드래그 앤 드롭으로 수십 장의 이미지를 한 번에 추가하고 변환할 수 있습니다.
- **🎚️ 정밀한 품질 제어**: 슬라이더를 통해 이미지 용량과 화질 사이의 최적점을 직접 실시간으로 설정할 수 있습니다.
- **📦 ZIP 다운로드 지원**: 변환된 이미지들을 개별적으로 받을 필요 없이, 클릭 한 번으로 압축 파일(ZIP)로 묶어 다운로드할 수 있습니다.
- **🎨 모던하고 직관적인 UI**: 복잡한 설정 없이 누구나 쉽게 사용할 수 있는 깔끔한 인터페이스를 제공합니다.

## 🛠️ 기술 스택

이 프로젝트는 최신 웹 기술을 기반으로 제작되었습니다.

- **Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Core Library**:
  - `jszip`: 클라이언트 측 ZIP 파일 생성

## 🚀 시작하기

### 설치 및 실행

1. 저장소를 클론합니다.
   ```bash
   git clone https://github.com/aklabs-84/webp_master.git
   ```
2. 의존성 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```
